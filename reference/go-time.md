# Go Time Reference

Go uses the specific numbers from Mon Jan 2 15:04:05 2006 as placeholders.

| Component | Reference | Meaning                   |
| --------- | --------- | ------------------------- |
| Year      | 2006      | 4-digit year              |
| Year      | 06        | 2-digit year              |
| Month     | Jan       | short month name          |
| Month     | January   | full month name           |
| Month     | 01        | zero-padded numeric month |
| Day       | 02        | zero-padded day           |
| Hour      | 15        | 24h hour                  |
| Hour      | 03        | 12h hour                  |
| Minute    | 04        | minute                    |
| Second    | 05        | second                    |
| AM/PM     | PM        | uppercase AM/PM           |
| AM/PM     | pm        | lowercase AM/PM           |
